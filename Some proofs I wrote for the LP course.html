Some Proofs written for the LP optimization

Date Sun 22 May 2022 By GZ Category Optimizatioon. Tags optimization

Here are some proofs I wrote for the optimization class.

An extreme point in a polyhedron is equivalent to a BFS

Let the polyhedron be \(P=\{x~|~ a_i^T x \le c_i, i=1,\dots, m, x \in R^n \}\).

  1. Extreme point => BFS

First, we show that at least one of the constraints \(a_i^T x \le c_i\) will have to be active. Let \(x_0\) be an extreme point and \(d\in R^n\) be some arbitrary direction.  Assume that \(a_i^T x < c_i\) for \(i=1,\dots, m\). Then I can always find a small enough \(\epsilon_i\) such that \(a_i^T (x_0 \pm \epsilon_i d) < c_i\). Let \(\epsilon = \min \{ \epsilon_1,\dots, \epsilon_m \}\), then we have \(a_i^T (x_0 \pm \epsilon d) < c_i\).  In other words, \(x_0\) is the midpoint of \((x_0 \pm \epsilon d)\), which contradicts with the fact that \(x_0\) is an extreme point.

Since some constraints will have to be active, let \(J\) be the set of indices of the active constraints such that \(a_j^T x_0 = c_j\) for \(j \in J\). Let \(A\) be a matrix whose rows are \(a_j\)  and \(c\) be a vector of \(c_j\) for \(j \in J\). By definition, \(A\) is a \(|J| \times n\) matrix and \(Ax_0=c\). Assume, to the contrary, that \(\text{null}(A) \ne \{0\}\). Let's take a nonzero vector \(d \in \text{null}(A)\). We see that \(A (x_0 \pm \epsilon d) =c\) or equivalently \(a_j^T (x_0 \pm \epsilon d) = c_j\) for any \(\epsilon \in R\). Also, by the logic of the previous paragraph, I can always find some \(\epsilon\) such that \((x_0 \pm \epsilon d)\) satisfies the remaining \(m - |J|\) constraints. Hence, \((x_0 \pm \epsilon d)\)  are in the polyhedron. Then again, \(x_0\) is the midpoint of \((x_0 \pm \epsilon d)\), which contradicts with the fact that \(x_0\) is an extreme point. This is to say that the \(\text{null}(A)=\{ 0\}\) and so \(Ax=c\) is a unique solution and thus BFS. 

  1. BFS => extreme point

Let \(x_0\) be the BFS such that it is the unique solution of \(\{x ~|~ a_j^T x = c_j, j \in J\}\) where \(|J| >0\). Now assume \(x_0\) is not an extreme point such that \(x_0 = \lambda x_1 + (1-\lambda) x_2\) for \(\lambda \in (0,1)\) and \(x_1 \ne x_2\) in the polyhedron.  Since \(x_0\) is the unique solution to \(\{x ~|~ a_j^T x = c_j, j \in J\}\), we find there exists at least a constraint \(k \in J\) such that \(a_k^T x_1 < c_k\), but then \(a_k^T x_0=a_k^T [\lambda x_1 + (1-\lambda) x_2 ]< c_k\), which contradicts with the fact that \(a_k^T x_0=c_k\). So \(x_0\) has to be an extreme point.

A nonempty polyhedron has an extreme point if and only if it does not contain a line

Let the polyhedron be \(P=\{x ~|~ a_i^T x \le c_i, i=1,\dots, m \}\) in \(\mathbb R^n\).

  1. If P contains a line then it does not have an extreme point

If \(m<n\), then there can be no unique basic solution and so there is no extreme point. Below we assume \(m\ge n\). 

Let the line it contains be\(b+\theta d\). Since the line cannot cross any hyperplane, we have to have \(a_i^T (b+\theta d) \le c_i\) for all \(\theta\). Hence, \(a_i^T d=0\). If we take any \(n\) hyperplanes, since \(d\) is orthogonal and thus linearly independent to the \(n\) normal vectors, the \(n\) vectors cannot be all linearly independent. Hence, there is no unique basic solution and so no extreme point. 2. "If P does not have an extreme point, then it contains a line"

P does not have an extreme point means that there is no basic solution, which in turn means that the list of normal vectors \(a_1, \dots, a_m\) has dimension less than \(n\). So there exists a vector \(d\) such that \(a_i^T d=0\) for \(i=1,\dots, m\). Since the polyhedron is nonempty, let \(b \in P\). Then we can easily verify that the line \(b +\theta d\) is in \(P=\{x ~|~ a_i^T x \le c_i, i=1,\dots, m \}\). 

If an LP has a finite optimal solution, then an optimal solution is a BFS (assuming the feasible region has a BFS)

This directly follows from the fact that a BFS is an extreme point. If an optimal solution \(x^*\) is not a BFS, then it can be written as some convex combination of the extreme points \(x^* = \sum_{i=1}^k \lambda_k x_k\). Since the objective function \(f(x)=c^Tx\) is concave (and convex), by concavity, \(f(x^*) \ge \sum_{i=1}^k \lambda_k f(x_k)\). Because \(x^*\) is optimal, \(f(x_k)=f(x^*)\).
